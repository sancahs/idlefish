Metric glossary

Analysis jobs
Completed fishnet work items found in local logs. These usually map to Lichess analysis tasks, so they are the closest local estimate of games helped.
Positions
Chess positions processed by fishnet. This is the easiest way to picture the useful chess work behind the raw CPU time.
CPU-hours
Estimated CPU time consumed by the fishnet service. One full CPU core running for one hour is about one CPU-hour.

idlefish is unofficial and community-run. Run spare compute responsibly.
